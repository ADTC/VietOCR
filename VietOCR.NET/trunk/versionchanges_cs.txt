VietOCR.NET Change Summary

Version 0.1 Beta (7 June 2008): 
- Proof-of-concept design
- Added post-processing for Vietnamese to improve accuracy
- Integrated Tessnet2 .NET wrapper for Tesseract 2.03 OCR engine
- Use beta Vietnamese language data for Tesseract 2.03 (data for 2.01 crashes frequently with Tesseract 2.03)
- Integrated VietKeyInput class library

Version 0.2 Beta (10 June 2008): 
- Improved post-processing for Vietnamese
- Added Help file

Version 0.3 Beta (21 June 2008): 
- Added file drag-n-drop support
- Added selection rectangle

Version 0.3.1 Beta (22 June 2008): 
- Made Picturebox scrollable with mouse wheel

Version 0.4 Beta (1 July 2008): 
- Code refactored
